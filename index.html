💡 System Bus Guide

Address Bus: Used by the CPU to specify the memory location it wants to read from or write to. In your project, this is visualized as the Address Signal (Blue).
Data Bus: The path through which the actual data (operands, results, instructions) travels between the CPU, memory, and registers. In your project, this is visualized as Data Flow (Green).
Control Bus: Carries command signals from the control unit to other components (like "Read", "Write", or "ALU Execute") to synchronize operations. In your project, this is visualized as Control Signal (Yellow).
